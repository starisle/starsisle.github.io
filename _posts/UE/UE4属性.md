## 核心数据类型

### 整数

整数数据类型转换是“int”或“uint”后跟位大小。

|  变量类型  |    说明    |
| :--------: | :--------: |
| **uint8**  | 8位无符号  |
| **uint16** | 16位无符号 |
| **uint32** | 32位无符号 |
| **uint64** | 64位无符号 |
|  **int8**  | 8位有符号  |
| **int16**  | 16位有符号 |
| **int32**  | 32位有符号 |
| **int64**  | 64位有符号 |
